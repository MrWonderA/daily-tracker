# 日常打卡 · 个人管理工作台

一款轻量化个人日常综合打卡 Web 网站，主打 **生活化记录、每日复盘、学习管理、个人记账、习惯养成** 五大核心体系。全程无社交、无广告、无推荐信息流，专注服务个人自律管理、生活留存、数据复盘。

> 纯前端静态页面，数据完全存储在本地浏览器，无需联网、无需注册、无需后端。

---

## 功能概览

### 📝 每日日记
- 富文本日志记录（文字、换行、简单标记）
- 8 种心情标签（开心、平淡、疲惫、焦虑、充实等），自动生成情绪曲线
- 每日复盘模板（今日完成 / 今日不足 / 明日计划）
- 本地图片上传插入
- 日历视图高亮标记，左右切换月份
- 关键词搜索、日期筛选、心情筛选
- 日志置顶、随手便签

### 📚 学习打卡
- 学习台账记录（科目、内容、时长、工具）
- 手动录入时长，自动累计日/周/月，生成柱状图
- 每日任务管理（添加、勾选、备注原因）
- 一键打卡签到，连续天数统计
- 书籍管理（书名、作者、进度、笔记、摘抄）
- 技能清单（练习进度、掌握程度、感悟）
- 内置专注计时器（正计时 / 倒计时）

### 💰 智能记账
- 收支记录（收入 / 支出 / 转账）
- 内置通用分类 + 用户自定义分类
- 日/周/月/年度账单视图一键切换
- 支出饼图、收支趋势折线图、月度对比柱状图
- 月度消费预算管控，超预算提醒
- 总资产 / 本月结余 / 净收入统计
- CSV 导出备份

### ✅ 习惯追踪
- 自定义习惯，支持 **四种量化追踪方式**：
  - 勾选（完成 / 未完成）
  - 时长（分钟）
  - 时间（具体时刻）
  - 次数
- 每日打卡勾选，生成完成率
- 月度热力图可视化
- 坏习惯自我监督（抽烟、熬夜等）

### 📊 数据仪表盘
- 全站数据聚合面板
- 核心统计卡片（日记数 / 学习时长 / 打卡天数 / 结余）
- 今日概览（待办 / 记账 / 学习 / 日记）
- 趋势图表（学习 / 情绪 / 收支 / 习惯）

### ⚙️ 系统设置
- 浅色 / 深色主题一键切换
- 数据手动备份导出 / 导入恢复
- 一键清除所有数据（含二次确认）
- 本地存储用量统计

## 快速开始

### 本地使用
直接双击 `index.html` 在浏览器打开即可，无需任何构建工具或服务器。

### 部署上线
项目为纯静态网站，支持任何静态托管服务：

```bash
# 1. 克隆仓库
git clone https://github.com/your-username/daily-tracker.git

# 2. 部署到任意静态服务器
# 例如：GitHub Pages、Vercel、Netlify、Nginx 等
# 将 daily-tracker/ 目录下所有文件上传至服务器根目录即可
```

**支持的部署平台：**
- **GitHub Pages** — 推送到 `gh-pages` 分支或项目 Pages 设置
- **Vercel** — 直接导入项目，自动识别静态站点
- **Netlify** — 拖拽文件夹或连接 Git 仓库
- **Nginx / Apache** — 将文件放置于 web 根目录
- **本地服务器** — `python -m http.server 8080`

## 设计理念

- **极简留白** — 低饱和度柔和配色，护眼舒适
- **傻瓜化操作** — 一键新增、一键查看、一键统计
- **高私密性** — 全部数据本地存储，不上传任何服务器
- **零干扰** — 无广告、无社交、无推荐流
- **全平台适配** — 移动端底部导航、桌面端侧边导航，严格响应式

## 隐私说明

- ✅ 无需联网、无需注册、无需手机号
- ✅ 全部数据存储在浏览器 LocalStorage
- ✅ 图片以 Base64 编码本地存储
- ✅ 支持手动导出备份文件
- ❌ 不收集任何个人信息
- ❌ 不上传任何数据到服务器
- ❌ 无第三方统计或跟踪代码
